Programación I - Ejercicio 02

Consigna

  • Crear e inicializar un array multidimensional para que almacene las notas que un/a estudiante obtuvo en el cuatrimestre.
  • Cada fila (array) representa una materia, y cada posición almacena una de las 3 notas que obtuvo en dicha materia.

  • La matriz se debería de ver algo así:

    notas = [
                            [4, 9, 10],   // Notas de la materia 1
                            [8, 5, 5],    // Notas de la materia 2
                            [7, 8, 5]     // Notas de la materia 3
                        ]

  • Se muestra al usuario un botón que al hacer clic debe de calcular e informar:

    • El promedio de las 3 notas obtenidas en cada materia.
    • La nota más alta entre todas las materias.
  • Punto extra 👑: Se muestra al usuario otro botón que al hacer clic debe de solicitar 3 notas (válidas del 1 al 10) y guardarla en el array de notas.
